## 📝 Sobre o Projeto
A aplicação consiste em uma interface web dinâmica projetada para resolver gargalos operacionais de ONGs do Terceiro Setor focadas no acolhimento de felinos. O sistema centraliza:
1. O cadastro e controle de status de saúde dos animais abrigados.
2. A escala de turnos dos voluntários.
3. Campanhas financeiras de arrecadação com barras de progresso interativas em tempo real.

## 🛠️ Tecnologias Utilizadas
* HTML5 Semântico: Para garantir acessibilidade estrutural e SEO.
* Tailwind CSS: Para estilização responsiva focada em dispositivos móveis (*Mobile First*).
* JavaScript (Vanilla): Para gerenciamento dinâmico de estados de dados e renderização de componentes na tela.
* Web Storage API (LocalStorage): Utilizada para emular a persistência de dados do banco de dados no navegador.

## 🚀 Como Executar o Projeto Localmente
Como o projeto foi consolidado em uma arquitetura de página única (SPA) autocontida, a execução é extremamente simples:
1. Faça o download ou clone este repositório.
2. Navegue até a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente em qualquer navegador moderno (Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, etc.).
4. Adicione dados nos formulários para testar a reatividade do sistema. Os dados permanecerão salvos mesmo se a página for atualizada.
